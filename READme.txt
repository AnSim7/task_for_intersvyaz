В папке parser находится исходный код программы разделения файла логов.
В папке web-interface находится исходный код пользовательсуого интерфейса программы, где пользователь может просматривать отчеты.
Database.sql - файл с конфигурацией базы данных.
запрос1.txt - запрос для "Посетители из какой страны совершают больше всего действий на сайте?".
запрос2.txt - запрос для "Посетители из какой страны чаще всего интересуются товарами из определенных категорий?".
запрос7.txt - запрос для "Какое количество пользователей совершали повторные покупки за определенный период?".
